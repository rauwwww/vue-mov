## Setup

### requisites

Node v10.16+
yarn

Install the dependencies and devDependencies and start the server.

### Run Frontend project

Build, Compiles and hot-reloads for development:

```sh
$ yarn
$ yarn serve
```

### Run Graphql serv Docker image

```sh
$ docker-compose up
```

### Lints and fixes files

```
yarn lint
```
