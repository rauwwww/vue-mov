## Setup

### requisites

Node v10.16+
yarn

Install the dependencies and devDependencies and start the server.

### Run project

Build, Compiles and hot-reloads for development:

```sh
$ yarn
$ yarn serve
```

### Run Apollo server

```sh
$ yarn apollo:start
```

### Lints and fixes files

```
yarn lint
```

### Test

Run end-to-end + unit tests

```sh
$ yarn test:e2e
$ yarn test:unit
```
